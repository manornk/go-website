# Go Website

Installation
------------

```bash
$ go get -u github.com/manornk/go-website
$ cd $GOPATH/src/github.com/manornk/go-website
$ go install .
```

Usage
-----

When everything is setup correctly you can run `go-website` with the following
command: 

```bash
$ cd $GOPATH/bin
$ go-website
```
